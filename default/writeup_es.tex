        % Explotación
        \section{Explotación}
        Durante la etapa de reconocimiento se determinó que el host objetivo ejecutaba \textbf{Samba 3.0.20}. Con esta identificación se realizó una búsqueda dirigida de vulnerabilidades y exploits relevantes mediante \textbf{searchsploit} (Metasploit), con el fin de seleccionar vectores de explotación aplicables a la versión encontrada.\vspace{0.2cm}        \vspace{0.2cm}
        \begin{figure}[h]
                \centering
                \includegraphics[width=.9\textwidth]{images/search-samba.png}
                \caption{Búsqueda de exploits para Samba 3.0.20 con searchsploit.}
        \end{figure}
        \vspace{0.2cm}
        \clearpage

        Se seleccionó el exploit asociado a la vulnerabilidad \textbf{CVE-2007-2447} (Samba 'username map script' Remote Code Execution), que permite la ejecución remota de comandos mediante la manipulación del parámetro \texttt{username map script} en la configuración de Samba. A continuación, se procedió a configurar el exploit con los siguiente parametros.\vspace{0.2cm}
        \begin{figure}[h]
                \centering
                \includegraphics[width=.9\textwidth]{images/show-options.png}
                \caption{Configuración y ejecución del exploit en Metasploit.}
        \end{figure}
        \vspace{0.2cm}

        Finalmente, se logró obtener una shell remota en el sistema objetivo, confirmando el éxito de la explotación y permitiendo el acceso al entorno del host comprometido.
        \begin{figure}[h]
                \centering
                \includegraphics[width=.9\textwidth]{images/exploitation.png}
                \caption{Obtención de shell remota en el sistema objetivo.}
        \end{figure}
